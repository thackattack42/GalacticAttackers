Galactic Attackers
=========================
A game made completely from scratch, including the rendering and Entity Component System (ECS)

Rules
-------------------------

- Shoot enemies with SPACE to deal damage
- Defeat enemies to gain points
- If an enemy touches the shield, they are destroyed but you get no points and your shield is used up
- If an enemy reaches the bottom of the screen, you lose a life. Lose all your lives and its Game Over
- Defeat all the enemies to win!

Features
-------------------------

- Level building in Blender and imported over to Visual Studio and rendered
- Player and enemies created using ECS
- Player, enemy, and projectile movement done using matrix math and physics
